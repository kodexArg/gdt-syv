# Valor

## Qué es el Valor

Tirada de salvación por moral. Cuando una unidad enfrenta una situación crítica, hace un chequeo de Valor para determinar si resiste o se quiebra. Incluso una tropa con moral baja (1 o 2) puede superar la tirada y permanecer controlada.

## Cuándo se tira

- Unidad fuera del radio de mando (regla de los 5 hexes)
